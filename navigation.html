Welcome to Home Page

Main Ingredients:

Wifi settings for the local device can be configured in this page. This page consist of WiFi Scan button, that scans and displays the near-by wifi networks. User can connect to the desired wifi network by entering SSID and password details.

Modbus TCP settings for the local device can be configured in this page. This page provides Modbus Read as well as Modbus Write options to read/write data to the end-device. User can read/write data to four function coils registers. Address, number of registers to be used need to be specified depending on the end-device

Mqtt broker settings can be configured in this page. User can connect to the public Mqtt brokers by entering Host, Port, ClientId, Username, and Password details.
